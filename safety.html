How We Protect Every Trade

1. Identity Verification

Every user must verify their identity before trading. We use phone number verification, email confirmation, and optional ID checks for high-value trades.

  • Phone number required
  • Email verification
  • Optional government ID for trades over $500
  • Social media linking for trust score

2. Escrow System

We hold both items until they're verified. No one gets anything until everyone is satisfied.

  • Both items must be shipped before trade completes
  • 48-hour inspection window after delivery
  • If something's wrong, we reverse the trade
  • Items shipped back at no cost if trade fails

3. Private Addresses

Your home address is never shared. We use secure proxy addresses so both parties stay anonymous.

  • Proxy shipping addresses only
  • No personal information shared
  • Optional Amazon Locker delivery
  • Your location stays private

4. Reputation System

Rate every trade. Build trust. Traders with better ratings get priority matching.

  • 5-star rating system
  • Detailed reviews (optional)
  • Trade history visible to matches
  • Report suspicious activity instantly

5. 24/7 Support Team

Real humans, ready to help. Average response time: under 2 minutes.

  • Live chat available 24/7
  • Dispute resolution within 24 hours
  • Phone support for urgent issues
  • Detailed help center with guides
